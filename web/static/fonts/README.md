# Локальные шрифты и ресурсы

Этот каталог содержит локальные копии шрифтов для работы без интернета.

## Структура:
- `inter.css` - CSS файл с @font-face правилами
- `ttf/` - файлы шрифтов Inter (Thin, Regular, Medium, SemiBold, Bold)

## Шрифты:
- Inter Thin (300)
- Inter Regular (400)
- Inter Medium (500)
- Inter SemiBold (600)
- Inter Bold (700)

Все шрифты загружаются локально, внешние подключения к Google Fonts отсутствуют.
